# 🔍 GitHub Analyser

> Enter any username. See what recruiters see.

A full GitHub profile analyser that fetches real data from the GitHub API and gives any profile an honest recruiter score.

---

## 🔥 Why I built this

I looked at my own GitHub and realised I had no idea what recruiters actually saw when they visited. Was my profile good? Bad? Embarrassing? I had no way to know.

So I built something to find out.

**Problem:** Developers don't know how their GitHub profile looks to a recruiter scanning it in 10 seconds.  
**Solution:** Enter any username and get a full breakdown — top languages, best repos, and an honest score out of 100.

> "Enter your username at your own risk. The truth about your commit history has been known to sting." 😂

---

## ✨ Features

- Fetches live data from the GitHub REST API
- Calculates a recruiter score out of 100
- Shows top languages with animated progress bars
- Lists top repos ranked by stars
- Full score breakdown so you know what to improve
- Beautiful gradient sidebar dashboard layout
- Dark and light mode

---

## 🛠️ Built with

- Vanilla JavaScript
- GitHub REST API
- Promise.all() for parallel API calls
- CSS3 with custom properties
- Array destructuring and optional chaining
